## 📐 Especificações Técnicas

### Formatos Aceitos:
- **Preferencial:** WebP (melhor compressão)
- **Alternativo:** JPG (fotos), PNG (logos/transparência)
- **Evitar:** BMP, TIFF (muito pesados)

### Tamanhos Recomendados:
| Tipo | Tamanho | Peso Máximo |
|------|---------|-------------|
| Logo | 200x60px | 50KB |
| Favicon | 32x32px | 10KB |
| Hero Images | 1920x1080px | 300KB |
| Service Images | 800x600px | 150KB |
| Blog Images | 1200x800px | 200KB |
| Thumbnails | 400x300px | 80KB |

### Otimização:
- **Qualidade JPG:** 80-85%
- **Compressão PNG:** TinyPNG ou similar
- **WebP:** Qualidade 80%
- **Responsive:** Criar versões mobile (@2x, @3x)

## 🎨 Diretrizes de Estilo

### Cores da Marca:
- **Vermelho:** #ef4444 (brand-red)
- **Preto:** #1a1a1a (brand-black)
- **Cinza:** #2d2d2d (brand-gray)
- **Branco:** #ffffff

### Estilo Fotográfico:
- **Tom:** Profissional, sério, confiável
- **Iluminação:** Natural ou bem iluminada
- **Composição:** Limpa, organizada
- **Foco:** Equipamento, segurança, qualidade

### O Que Evitar:
- ❌ Fotos desfocadas ou mal iluminadas
- ❌ Imagens genéricas de banco de imagens
- ❌ Fotos com marcas d'água
- ❌ Imagens com baixa resolução
- ❌ Fotos que não representam o serviço real

## ✅ Como Adicionar Imagens

### Método 1: Via GitHub Web Interface
1. Acesse: https://github.com/dayrtonc/DL-Demolition-and-Asbestos-experts
2. Navegue até `assets/images/`
3. Clique em "Add file" > "Upload files"
4. Arraste as imagens
5. Commit com mensagem descritiva

### Método 2: Via Git Command Line
```bash
# Clone o repositório (se ainda não tiver)
git clone https://github.com/dayrtonc/DL-Demolition-and-Asbestos-experts.git
cd DL-Demolition-and-Asbestos-experts

# Adicione as imagens
cp /caminho/para/suas/fotos/* assets/images/

# Commit e push
git add assets/images/*
git commit -m "feat: Add professional service photos"
git push origin main
```

### Método 3: Via FTP (se tiver acesso ao servidor)
1. Conecte via FTP ao servidor
2. Navegue até `assets/images/`
3. Upload das imagens
4. Verificar permissões (644)
